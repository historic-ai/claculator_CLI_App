
#  Calculator CLI App

A simple command-line calculator built using Python.

##  Features
- Addition, Subtraction, Multiplication, Division
- Handles invalid input and division by zero
- Loops until user chooses to exit

##  Tools Used
- Python
- VS Code / Terminal
